Hotel Revenue Analysis

The above dashboard was developed using Microsoft's Power BI to visualize Eskom's Power generation and you can actually interact with it! The dashboard expresses Eskom's Infrastructural data from multiple power stations, this includes capacity, location, ect. It also expresses the capacity and efficiency of each power station. The dashborad also draws data from multiple tweets from the officail Eskom's Tweeter page. The dashboard can be used to derive insights towards tackling loadshedding.
